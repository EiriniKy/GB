## Τεχνική Αναφορά - Λογισμικό Government Budget
### 1. Περιγραφή Λογισμικού
Τοπική εφαρμογή που επιτρέπει στον χρήστη την προβολή και επεξεργασία του κρατικού προϋπολογισμού.
Κύριες λειτουργίες:
 * Προβολή των στοιχείων του κρατικού προϋπολογισμού.
 * Επεξεργασία των στοιχείων του κρατικού προϋπολογισμού.
 * Δυνατότητα σύγκρισης του κρατικού προϋπολογισμού της Ελλάδας του έτους 2025 με τους κρατικούς προϋπολογισμούς των αντίστοιχων χωρών: Ολλανδία, Γερμανία, Ισπανία. 
 * Δυνατότητα σύγκρισης του κρατικού προυπολογισμού του έτους 2026 με τους κρατικούς προϋπολογισμούς των αντίστοιχων ετών: 2025, 2024, 2023.
 * Προβολή του Ιστορικού αλλαγών.
### 2. Οδηγίες Μεταγλώττισης
Το λογισμικό  γράφτηκε σε *Java 25*.

Η βάση δεδομένων γράφτηκε σε *Sqlite*

Για την κατασκευή της χρησιμοποιείται *Maven*.

Τα αρχεία πηγαίου κώδικα βρίσκονται στον φάκελο

#### Μεταγλώττιση από Command line, στον φάκελο του project:
* Εντολή: mvn clean package
### 3. Οδηγίες Εκτέλεσης
#### Από Command Line:
* Εντολή: java -jar target/Project-1.0-SNAPSHOT.jar

### 7. Δομές Δεδομένων και Αλγόριθμοι

- Χρησιμοποιούνται **ArrayList** για την αποθήκευση και διαχείριση συλλογών αντικειμένων.
- Χρησιμοποιούνται **HashMap** για γρήγορη αναζήτηση δεδομένων με κλειδί.
- Η πρόσβαση στα δεδομένα γίνεται μέσω **SQLite** χρησιμοποιώντας **JDBC**.
- Υλοποιούνται βασικές λειτουργίες **CRUD** (Create, Read, Update, Delete).
- Για την επεξεργασία δεδομένων εφαρμόζονται απλοί αλγόριθμοι:
    - γραμμική αναζήτηση
    - επανάληψη μέσω βρόχων (for / while)
